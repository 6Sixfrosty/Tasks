1. Remover o campo de observações.
2. Oferecer ao usuário a escolha de **um polo** entre duas opções:
   - Senai Santo Amaro
   - Senai Areias
3. Cada polo deve ter **5 cursos diferentes** disponíveis para seleção.
4. O usuário deve escolher **um polo** e **um curso correspondente ao polo**.
5. Ao enviar o formulário:
   - Enviar um **e-mail para o usuário**.
   - O e-mail deve conter:
     - Uma **imagem do curso escolhido**
     - Um **texto explicativo sobre o polo e o curso**

**Requisitos técnicos:**

- O formulário deve ser HTML com `<select>` para polos e cursos.
- O envio será feito via **POST** para um backend em **Python/Flask**.
- O backend deve gerar o e-mail automaticamente, preenchendo os dados do curso e polo selecionados.
